# Typography

## 字体

默认使用系统字体：

```css
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Microsoft YaHei", sans-serif;
```

数字建议使用表格式数字：

```css
font-variant-numeric: tabular-nums;
```

## 字号体系

| 名称 | 字号 / 行高 | 字重 | 用途 |
| --- | --- | --- | --- |
| H1 | 28 / 36 | 700 | 少用，重大页面标题 |
| H2 | 24 / 32 | 700 | 页面标题 |
| H3 | 20 / 28 | 700 | 页面分区标题 |
| H4 | 16 / 24 | 600 | 卡片标题 |
| H5 | 14 / 22 | 600 | 表格区域标题 |
| H6 | 12 / 20 | 600 | 小标题 |

## 页面标题

页面标题：

- 24px。
- 700 或 760。
- 不超过 12 个中文字符。
- 不使用负字距。

页面说明：

- 14px。
- 次级色。
- 1 到 2 行。

## 模块标题

模块标题：

- 16px。
- 600-700。
- 与上方模块保持 24px 间距。

模块说明：

- 14px。
- 次级色。
- 不写操作教程，只说明模块用途。

## 正文

正文：

- 14px / 22px。
- 主文本色。

说明文字：

- 12px / 18px。
- 次级色或弱提示色。

## 数据展示文字

KPI Value：

- 24px。
- 700。
- 使用表格式数字。

表格数字：

- 14px。
- 右对齐。
- 表格式数字。

状态标签：

- 12px。
- 600。

## Typography 检查项

- 标题是否不过大。
- 数字是否便于比较。
- 说明文字是否不超过 2 行。
- 是否没有负字距。
