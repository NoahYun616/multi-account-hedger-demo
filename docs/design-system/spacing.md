# Spacing

## 间距 Token

统一采用以下间距：

| Token | Value | 用途 |
| --- | --- | --- |
| `space.4` | 4px | 图标与文字、紧密元素 |
| `space.8` | 8px | 表单项内部、标签间距 |
| `space.12` | 12px | 小卡片内部、按钮组 |
| `space.16` | 16px | 卡片内边距、表格区块 |
| `space.24` | 24px | 模块间距、页面主要分区 |
| `space.32` | 32px | 大区块间距 |
| `space.48` | 48px | 页面级大间距，少用 |

## 页面间距

- 主内容左右边距：24px。
- 页面顶部：16-24px。
- 页面模块间距：24px。
- KPI 卡片间距：16px。
- 表格与标题间距：8-12px。

## 卡片间距

- 卡片内边距：16px。
- 大卡片内边距：24px。
- 卡片之间：16px 或 24px。
- 卡片圆角：16px。

## 表单间距

- Label 与控件：8px。
- 表单项之间：16px。
- 表单分组之间：24px。
- 按钮组与表单：24px。

## 表格间距

- 表头单元格内边距：12px 16px。
- 表体单元格内边距：12px 16px。
- 表格上方工具栏与表格：12px。
- 分页与表格：16px。

## 响应式间距

窄屏：

- 页面左右边距降到 16px。
- KPI 卡片间距 12px。
- 大区块间距 16-24px。

## Spacing 检查项

- 是否只使用指定间距 Token。
- 模块之间是否清晰分组。
- 卡片内边距是否一致。
- 移动端是否没有过大留白。
