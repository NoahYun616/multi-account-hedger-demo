# 合约保险对冲工具迭代文档

本目录用于存放合约保险对冲工具后续优化升级过程中的产品方案、研发约定、项目状态和迭代资料。

## 当前文件

- `合约保险对冲工具优化迭代方案.md`：当前优化升级的主方案。
- `agent.md`：给后续接手该工具的 AI / 研发协作者看的工作约定。
- `project_state.md`：当前项目状态、已完成事项、待办和关键决策记录。
- `git_workflow.md`：工具迭代的 Git 管理方式。
- `frontend_style_guide.md`：前端风格草案，确认后作为后续页面和功能开发的视觉与交互标准。

## 使用规则

- 每次开始工具迭代前，先阅读 `project_state.md` 和主方案。
- 如果本次迭代改变了工具能力、启动方式、配置方式或风险提示，需要同步更新工具根目录的 `README.md`。
- 新增产品方案、接口说明、测试方案、验收记录等文档，优先放在本目录下。
- 如果文档内容与代码实现不一致，以代码和最新 `project_state.md` 为准，并及时修正文档。
- 工具已使用独立 Git 仓库管理；提交前需确认真实账号配置、日志和虚拟环境没有进入版本管理。
